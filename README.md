## Requirements

- [Node.js](https://nodejs.org/en/) >= v16.18.0
- [Yarn](https://yarnpkg.com/en/) >= 1.22.19
- [Npm](https://www.npmjs.com/) >= 8.19.2

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn dev

# production mode
$ yarn run start:prod
```
